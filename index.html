Specifications

Property4U core is based on ASP.NET MVC 5 ASP.NET, WEB API 2.2, ASP.NET Identity 2.0, ASP.NET SignalR, Bootstrap 3, EntityFramework 6, Json.NET and Quartz.NET packages.

With HTML 5, CSS 3, jQuery, LESS CSS, Twitter Bootstrap, FontAwesome & Co, KnockoutJS offers designers a foundation for creating responsive layouts, thereby enabling real estate designs which are platform independent.

The cutting-edge architecture of Property4U - with ASP.NET 4.5 + MVC 5, WEB API 2.2, Entity Framework 6 and MS SQL Server - makes it extendable and highly flexible.

Features

  • Open Source

    Property4U is an 100% Open Source under Microsoft Public License with no hidden cost.

  • Responsive Design

    It's browser friendly design makes system UI compatible with any web or smart phone platform.

  • Configurable

    It allows any organization to configure system as per there business needs.

  • WEB API

    The powerful Documented Web API provides read and write access to system resources base on user role.

  • SEO Optimized

    Property4U takes care of search engine friendly URLs, modern HTML-Layout, Meta-Tags...

  • New Technology

    The cutting-edge architecture of Property4U - with ASP.NET 4.5 + MVC 5, WEB API 2.2, Entity Framework 6 and MS SQL Server - makes it extendable and highly flexible.

  • User Management

    Makes user and role management simpler and flexible.

  • Sale/Rent

    Streamline the daily life sale/rent property process with ease.

  • Bidding

    Property4U brings you the proficient bidding features that is missing in most of Real Estates system.

  • Order

    In Integrated ControlDesk, you can manage orders.

  • Advertisement

    The Integrated ControlDesk, lets you manage ads base on requirements.
